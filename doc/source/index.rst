Welcome to API-Test's documentation!
====================================

API-Test is a collection of tools that allow to test in many ways HTTP APIs

+----------------+------------------------------------+
|Project site    | http://github.com/bbva/apitest     |
+----------------+------------------------------------+
|Documentation   | http://apitest.readthedocs.org     |
+----------------+------------------------------------+
|Author          | Daniel Garcia (cr0hn) - @ggdaniel  |
+----------------+------------------------------------+
|Last Version    | 1.0.0                              |
+----------------+------------------------------------+
|Python versions | 3.5 or above                       |
+----------------+------------------------------------+

Some of API Test features:

   - Importing APITest collections
   - Building black-box tests cases as unittest format (useful to integrate in a C.I. software).
   - Analyzing APIs files / documentation
